How to U se Selectors?

The selectors are very useful and would be required at every step while using jQuery. They get the exact element that you want from your HTML document. Following table lists down few basic selectors and explains them with examples. S.N. Selector & Description 1 Name Selects all elements which match with the given element Name. 2 #ID Selects a single element which matches with the given ID. 3 .Class Selects all elements which matches with the given Class. 4 Universal (*) Selects all elements available in a DOM. 5 Multiple Elements E, F, G Selects the combined results of all the specified selectors E, F or G. jQuery - Element Name Selector Description The element selector selects all the elements that have a tag name of T.

Syntax
