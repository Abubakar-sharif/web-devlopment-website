AngularJS Directives The AngularJS framework can be divided into three major parts:  ng-app : This directive defines and links an AngularJS application to HTML.  ng-model : This directive binds the values of AngularJS application data to HTML input controls.  ng-bind : This directive binds the AngularJS application data to HTML tags.
